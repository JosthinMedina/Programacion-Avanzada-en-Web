# Sistema de Gestión Académica - Escuela Aurora

## Descripción

Sistema desarrollado para el curso **Programación Avanzada Web** de la Universidad Fidélitas.

El proyecto tiene como objetivo automatizar los procesos académicos de la Escuela Aurora mediante una aplicación web basada en ASP.NET Core MVC y una Web API.

## Tecnologías

- C#
- ASP.NET Core MVC
- ASP.NET Core Web API
- SQL Server
- Dapper
- JWT
- Bootstrap 5
- Razor
- Git & GitHub

## Funcionalidades

- Registro de usuarios
- Inicio y cierre de sesión
- Recuperación de contraseña
- Gestión de estudiantes
- Gestión de profesores
- Gestión de cursos
- Matrícula de estudiantes
- Registro de calificaciones
- Registro de asistencia
- Calendario académico
- Reportes
- Notificaciones por correo electrónico
